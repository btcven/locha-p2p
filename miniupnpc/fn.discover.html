pub fn discover(
    delay: Duration, 
    multicast_if: Option<&str>, 
    minissdp_sock: Option<&str>, 
    localport: LocalPort, 
    ipv6: bool, 
    ttl: u8
) -> Result<DeviceList, Error>

Discover uPnP devices.

This function blocks until all devices have been discovered. delay impacts the waiting time.

Arguments

  • delay: Maximum delay for a device to respond.
  • multicast_if: Multicast network interface to use if provided.
  • minissdp_sock: Minissdpd sock if provided.
  • localport: Local port (source) to use.
  • ipv6: use IPv6.
  • ttl: sets the value of IP_MULTICAST_TTL or IPV6_MULTICAST_HOPS respectively depending on which protocol is used. Default value is 2.

Errors

This function might return an error on Out-of-memory or when the socket fails.
